Compute the complete fit over all orders.

Computes the fit of polynomials of varying order to a given dataset. Used to determine the relative likelihood of support for polynomials of different order to a given dataset.

Parameters
cfThe curvefit_result_t object
dataThe dataset
diFirst index into dataset
djLast index into dataset
max_orderThe maximum order polynomial to fit (cannot be greater than that of th the curvefit object).
fixed_priorUse a fixed prior for the different orders (set to NULL to use the automatic prior).
auto_zZ score for automatic prior, ie 3 = 3 standard deviations prior.
mean_misfitOutput array of the misfit of the mean curve to the data for each order.
detCmOutput array of determinant of the Covariance matrix for each order.
SCovariance Matrix output
pkOutput array of probability of each order.
kcdfCummulative probability of each order.
meanOutput Mean curves for each order.
sigmaOutput Standard deviation for each order.
